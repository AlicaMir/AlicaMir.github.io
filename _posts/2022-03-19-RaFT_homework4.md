#### a) a = 7 + i

$$N(a) = 7^2 + 1^2 = 50 = 5^22 $$ 

Попробуем разложить $$a$$ в произведение элементов простой нормы (они неразложимы). Какие элементы имеют норму 2? $$1\pm i$$

Проверим делимость: $$(7 + i, 1 + i) = (6, 1 + i) = (3(1 + i)(1 - i), (1 + i)) = (1 + i)$$, поэтому $$7 + i$$ делится на $$1 + i$$. Поделим. получим $$3(1 - i) + 1 = 4 - 3i$$. Какие элементы имеют норму 5? $$2\pm i$$ и $$1\pm 2i$$. Поделим $$(4 - 3i, 2 + i) = (10, 2 + i) = (2(2 + i)(2 - i), 2 + i) = (2 + i)$$, поэтому $$4 - 3i$$ делится на $$2 + i$$, получим $$2(2 - i) - 3 = 1 - 2i$$. Таким образом, 

$$7 + i = (1 + i)(2 + i)(1 - 2i)$$

Таким образом, $$11 + 2i = i(1 + 2i)(2 + i)(1 - 2i)$$.

### Задание 2

#### Докажите следующие факты про числа Эйзенштейна:

а) $$\mathbb{Z}[w] = \frac{\mathbb{Z}[x]}{x^2 + x + 1}$$

Определим $$\phi: \mathbb{Z}[x] \rightarrow \mathbb{Z}[w]$$ по правилу "$$a_0 + a_1x + ... + a_nx^n \mapsto a_0 + a_1w + ... + a_nw^n = A + Bw$$. Отображение сюрьективно: прообразом $$A + Bw$$ является $$A + Bx$$. Заметим, что ядром этого отображения являются многочлены, делящиеся на $$x^2 + x + 1$$. Действительно, раз многочлен при подстановке в него $$w$$ занулился, то $$w$$ - его корень, но тогда и $$-w$$ - корень, поскольку коэффициенты этого многочлена целочисленные. Ну а многочлен с такими корнями обязан делиться на $$(x - w)(x + w) = x^2 + x + 1$$ по теореме Безу.

Осталось показать, что $$\phi$$ гомоморфизм. Но теперь это просто: пусть $$f, g \in \mathbb{Z}[x]$$. Поделим их с остатком на $$x^2 + x + 1$$. Получим $$f = f_0 + f_1x + (x^2 + x + 1)F(x), g = g_0 + g_1x + (x^2 + x + 1)G(x)$$. Тогда $$\phi(f + g) = \phi((f_0 + g_0) + (f_1 + g_1)x + (x^2 + x + 1)(F + G)) = (f_0 + g_0) + (f_1 + g_1)w = (f_0 + f_1w) + (g_0 + g_1w) = \phi(f) + \phi(g)$$. А так же $$\phi(fg) = \phi(f_0g_0 + (f_0g_1 + f_1g_0)x + f_1g_1x^2 + (x^2 + x + 1)(Gf + gF)) = f_0g_0 + (f_0g_1 + f_1g_0)w + f_1g_1w^2 = (f_0 + f_1w)(g_0 + g_1w) = \phi(f)\phi(g)$$ и $$\phi(1) = 1$$.

По теореме о гомоморфизме получаем, что $$\mathbb{Z}[w] \cong \frac{\mathbb{Z}[x]}{\text{Ker}\phi} \cong \frac{\mathbb{Z}[x]}{x^2 + x + 1}$$.

### Задание 3

#### Норма в кольце чисел Эйзенштейна приводит нас к следующему вопросу: когда простое число p представляется в виде a^2 − ab + b^2?

а) Если $$p = 3k + 2$$, то $$p$$ неразложим в кольце $$\mathbb{Z}[w]$$.

От противного, пусть разложим. $$p = ab$$. $$N(p) = p^2 = N(a)N(b)$$. Если для какого-то из чисел норма равна 1, то это один из обратимых элементов $$\pm 1, \pm i$$. Тогда $$N(a) = p, N(b) = p$$. Поэтому для каких-то чисел x, y выполнено $$x^2 + y^2 - xy \equiv 2 (\mod 3)$$. Т.е. $$x^2 + y^2 - xy + 3xy - 3xy  \equiv (x + y)^2 \equiv 2 (\mod 3)$$, но $$\left(\frac{2}{3}\right) = \left(\frac{-1}{3}\right) = (-1)^{\frac{3-1}{2}} = (-1)^1 = -1$$, поэтому сравнение решений не имеет. Противоречие, значит $$p$$ неразложим.

б) Если $$p = 3k + 1$$, то $$p$$ разложим в кольце $$\mathbb{Z}[w]$$.

Заметим, что поскольку p простое, то $$k = 2t$$ для какого-то целого $$t$$. $$\left(\frac{-3}{p}\right) = \left(\frac{-3}{3k + 1}\right) = \left(\frac{-1}{6t + 1}\right)\left(\frac{3}{6t + 1}\right) = (-1)^{\frac{(6t + 1)-1}{2}}(-1)^{\frac{(6t + 1)-1}{2}}(-1)^{\frac{3-1}{2}}\left(\frac{6t + 1}{3}\right) = \left(\frac{1}{3}\right) = 1$$.

Поэтому $$-3$$ явялется квадратичным вычетом по модулю $$p = 3k + 1$$, т.е. $$\exists x: x^2 \equiv -3 (\mod p)$$, а значит $$x^2 + 3$$ делится на $$p$$.

$$x^2 + 3 = (x - 2w - 1)(x + 2w + 1)$$. Предположим, что $$p$$ неразложимо. Поскольку $$\mathbb{Z}[w]$$ евклидово, то тогда p прост в нем, а значит делит одну из скобок. Поскольку p - число, то оно делит вещественную и мнимую часть числа. У обоих чисел в скобках мнимая часть равна 2. Значит, $$p \mid 2$$. Поскольку 2 неразложим по пункту а), то $$p \sim 2$$ или $$p \sim 1$$, т.е. $$p = 2$$ или $$p = 1$$ Оба исхода дают противоречие. Получается, $$p$$ разложим.

### Задание 4

####  Простые числа вида 8k − 1, 8k − 3 неразложимы в Z[√2i].

Утверждение: Нормой в $$\mathbb{Z}[\sqrt 2i]$$ является $$N(a + \sqrt{2}b) = (a + \sqrt 2ib)(a - \sqrt 2ib) = a^2 + 2b^2$$. 

Пусть $$p$$ разложимо, тогда $$N(p) = p^2 = N(a)N(b)$$. Если хотя бы одна из норм $$a$$ и $$b$$ равна 1, то это один из обратимых элементов $$\pm 1$$. Иначе обе нормы равны $$p$$. 

Рассмотрим сравнение по модулю 8: $$a^2 + 2b^2 \equiv p (\mod 8)$$. Каковы квадратичные вычеты по модулю 8? Это $$(\pm 1)^2, (\pm 2)^2, (\pm 3)^2, 4^2$$, т.е. 1, 4 ну и можно приписать к ним 0. Всевозможные комбинации: $$4 + 2*1, 4 + 2*0, 4 + 2*4, 1 + 2*0, 1 + 2*1, 1 + 2*4, 0 + 2*0, 0 + 2*1, 0 + 2*4$$. Видим, что ни $$-1 \equiv 7$$, ни $$-3 \equiv 5$$ здесь нигде не достигается, а значит числа $$p = 8k - 1$$ и $$p = 8k - 3$$ неразложимы, т.к. пришли к противоречию.

### Задание 5

Рассмотрим кольцо $$\mathbb{Z}[\delta]$$, где $$\delta = \frac{-1 + \sqrt{19}i}{2}$$. Так же определим $$\overline{\delta} = \frac{-1 - \sqrt{19}i}{2}$$ и $$x$$ - необратимый элемент этого кольца с наименьшей нормой относительно мультипликативной нормы $$N$$.

Рассмотрим многочлен $$x^2 + x + 5$$. Его дискриминант: $$D = 1^2 - 4*5 = -19$$, тогда его корни $$\frac{-1 \pm \sqrt{19}i}{2}$$, т.е. $$\delta$$ и $$\overline{\delta}$$. По теореме Виета: $$\delta + \overline{\delta} = -1$$, т.е. $$\overline{\delta} = -1 - \delta$$. Значит, $$\overline{\delta}$$ элемент кольца $$\mathbb{Z}[\delta]$$ и этот многочлен можно разложить в произведение (по теореме Безу): $$x^2 + x + 5 = (x - \delta)(x + 1 + \delta)$$, где теперь x - то самое, что мы определили выше.

Ещё одно замечание: x неразложим. Действительно, пусть разложим, тогда представим его в виде $$x = ab$$, где $$a$$, $$b$$ - необратимы. Если $$N(a) < N(x)$$, то $$a$$ необратимый элемент с меньшей нормой, противоречие с выбором $$x$$. $N(a) > N(x)$$ не может быть, остается случай $$N(a) = N(x)$$. Поделим $$a$$ на $$x$$ с остатком: $$a = qx + r$$, где либо $$r = 0$$ (и тогда $$a \sim x$$, противоречие), либо $$N(r) < N(x)$$, поэтому $$r$$ - обратим. Преобразуем: $$a = qx + r = qab + r$$, тогда $$r = a(1 - qb)$$. Значит, $$1 = a(1 - qb)r^{-1}$$, поэтому $$(1 - qb)r^{-1}$$ обратный к $$a$$ - противоречие.

Поскольку кольцо евклидово, то неразложимый элемент прост, а идеал, порожденный простым элементом, максимален. Рассмотрим фактор-кольцо $$\frac{\mathbb{Z}[\delta]}{(x)}$$. Поскольку $$(x)$$ максимален, то это поле. Зададимся вопросом: делится ли 5 на $$x$$? Поделим $$x^2 + x + 5$$ на x с остатком: $$(x+1)x + 5 = x^2 + x + 5 = qx + r$$, где либо $$r = 0$$ (тогда делится), либо $$N(r) < N(x)$$, поэтому $$r$$ обратим. Взяв фактор по левой и правой части равенства, получаем, что 5 и r в одном классе ассоциированности.
